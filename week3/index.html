Getting data from the user

There exist three ways of getting data from the user using pure JavaScript.

  • alert(message): Shows the user the message, then waits until the user clicks "OK".
  • confirm(message): Shows the user the message, then waits until they click "OK" or "Cancel".
  • prompt(message, placeholder): Shows the user the message, allows them to write in a single-line text box that is pre-filled with placeholder, and waits until they click "OK" or "Cancel".

The Math Object

  • Math.sqrt(number): Returns the square root of the given number. Helpful for all that math I learned in High School and forgot.
  • Math.random(): Returns a random number between 0 and 1. Useful when you need an element of randomness, like in a game.
  • Math.round(number): Returns the next nearest integer. Great for rounding numbers when displaying values to users.

The <noscript /> tag

  • The <noscript> tag is an HTML tag that will hide whatever content is inside of it when JavaScript is turned on, and show that content whenever it is turned off.
  • This is important, as every modern browser gives you the ability to turn off JavaScript.
    • In Firefox: Tools > Options > Content (Tab), uncheck "Enable JavaScript"
    • In Internet Explorer (7+): Tools > Internet Options > Security (Tab) > Custom level... > Scripting > Active Scripting, select "Disable"
  • By including the <noscript> tag and content inside, you can either alert people to turn on JavaScript, or show content that doesn't require JavaScript.

Logical Operators

  • && and 
    var animal = "leopard", wild = true;
if (animal === "leopard" && wild === true) {
	// This block will be executed only if animal is leopard and wild is true
	alert("The animal is a Wild Leopard!");
}
  • || or 
    var animal = "leopard";
if (animal === "leopard" || animal === "tiger") {
	// This block will be executed if animal is either leopard or tiger
	alert("The animal is a Leopard or Tiger!");
}
  • ! not 
    var isAdmin = false;
if (!isAdmin) {
	// This block will be executed only if isAdmin is set to false
	alert("You are not an Admin!");
}

Switch Statement

  • Based on how the variable being 'switched' matches 'cases', different pieces of code are run
  • Similar to if() statement 
    switch(animal) {
	case "panther":
		alert("It's a Panther!");
		break;
	case "tiger":
		alert("It's a Tiger!");
		break;
	default:
		alert("It's some sort of animal!");
		break;
}
  • Allows a default, much like 'else' in an if() statement, but does not require it
  • Allows code execution to flow from one 'case' to the next, unless a 'break' statement is present
  • Looks a lot cooler than the if statement, when used in moderation

Loops

  • Loops allow you to keep running the same block of script over and over again, until some condition is met.
  • There are two types of loops in JavaScript:
    • for loop, which loops through a block of script a set number of times.
    • while loop, which loops through a block of script until a specified condtion is true.

for loop

  • The for loop is used when you know in advance how many times the script should run. #
  • The syntax is a bit difficult to look at (so keep this as reference for a while), but it makes sense: 
    var numTimes = 10;
for( var index = 0; index <= numTimes; index++ ) {
	alert(index + " Times!");
	alert(index + " Times!");
}
  • The three statements in the for loop's call are:
    • Initializer, which runs one when the loop starts, typically declaring a counter variable and setting it to the desired starting value
    • Loop-test, which runs at the beginning of every loop and, if it evaluates to false, the loop ends
    • Counting expression, which runs at the end of every loop, and is typically used to increment the counter variable

for...in loop

  • A modified version of the for loop is the for...in loop
  • It allows you to iterate over all of the values in an object or array
  • It's syntax is simpler, and quite useful 
    var browsers = {
	firefox: {title: "Mozilla Firefox", version: "4.0"},
	opera: {title: "Opera", version: "10.11"},
	safari: {title: "Safari", version: "5.0"},
	ie: {title: "Microsoft Internet Explorer", version: "9"}
};

for(var key in browsers) {
	alert(key + " version: " + browsers[key].version);
}
  • Do note that, unlike other languages, the key variable is set to the object key (firefox, opera, etc.) rather than the value ({title: "Mozilla Firefox", version: "4.0"}, etc.).

while loop

  • The while loop can make some tasks much simpler than using a for loop, especially when the number of times you need to loop is unknown 
    var number;
var valueIsNumeric = false;
while(!valueIsNumeric) {
	number = prompt("Please enter a number.");
	
	number = parseFloat(number); // Parses a float (decimal number) from the string
	
	if(!isNaN(number)) { // Check if the value is Not a Number (NaN)
		valueIsNumeric = true;
	}
}

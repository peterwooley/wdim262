Midterm Practical

To complete the Practical portion of the WDIM262 Midterm, please write a script that accomplishes the following:

  1. Ask the user which Web browser is their favorite (of three options you define), accepting both the full title or numeric position in the list you define,
    • i.e., if you present the user with "Choose: IE, Firefox, or Safari", the user can either enter "Firefox" or "2" to choose Firefox.
  2. Continue to ask the user which browser is their favorite, until a valid answer has been given.
  3. Dynamically create an <h1> tag with the title of the browser they chose,
  4. Dynamically create an <img> tag in the document that features the user's chosen Web browser logo.
  5. Lastly, no content may appear on the page if JavaScript is disabled in the user's browser, except for a message telling the user they should enable JavaScript to view the content.

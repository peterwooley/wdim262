Midterm Practical

To complete the Practical portion of the WDIM262 Midterm, please write a script that accomplishes the following:

  1. When the user clicks the calculte button, ask their name and age.
  2. Keep asking until they enter a value for both their name and an age.
  3. Lastly, show them their name, age and how many more years they'll live (this shoudl be random). Put this info in the results element.
  4. Note: See the following slide for the code you can start with.

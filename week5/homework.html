Requirements

Create a Web page that uses event handlers and the DOM to interact with the user in a few specific ways. Make sure it fulfills these requirements:

  • When the page loads, the user is greeted in some way.
  • When a user clicks on a link on the page the link text changes and the page does not go to the link's href location.
  • When a user's mouse enters a <div /> defined on the page, the div moves to random spot on the page.
  • Lastly, when the user resizes the window, the new width and height of the window (in pixels) are shown to the user in some way.

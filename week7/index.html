Traversing the DOM

The DOM is a tree data structure.

From any element in the DOM, you can use properties to get to any other element.

An element can have any of the following properties:

  • .parentNode
  • .childNodes — Returns an array of elements.
  • .lastChild
  • .firstChild
  • .previousSibling
  • .nextSibling

Getting content without .innerHTML

In JavaScript, text inside of an element is considered its own "node", a "Text node".

Knowing this, you can access text in a Text Node with the .nodeValue property.

var header = document.getElementById("header");

for(var i = 0; i !== header.childNodes.length; i++) {
    var node = header.childNodes[i];
    alert("Type: " + node.nodeType + ", Value: " + node.nodeValue);
}

Requirements

Create a Web page that uses event handlers and the DOM to interact with the user in a few specific ways. Make sure it fulfills these requirements:

  • When the page loads, the user is greeted in some way.
  • When the page unloads, the user is bid farewell in some way.
  • When a user clicks on a link on the page the link text changes and the page does not go to the link's href location.
  • When a user's mouse enters a <div /> defined on the page, the div's background color changes.
  • When a user's mouse leaves the afore-mentioned <div />, the background color changes back.
  • Lastly, when the user resizes the window, they are alert()ed that their action was successful.

Requirements

Create a form that can dynamically alter the page. Make sure it fulfills these requirements:

  • Using a text box, the user can enter a hex or CSS color name and the background color of the page will immediately change.
  • Using a series of radio buttons, the user can set the page font size to small, medium, and large.
  • The user can edit text in a textarea and immediately see the changes in a styled paragraph next to the textarea.
  • The user can enter JavaScript into a textbox and after clicking a button, the JavaScript is executed. (investigate the eval() function).

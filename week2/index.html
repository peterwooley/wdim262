Symbol Meaning

  • Unary operators require only one value to perform an action, rather than two.
  • Unary operators in JavaScript update the values of the variables on which they are used. 
    var num = 42;
num++; // Increments num variable by 1
num = num + 1; // Also increments num variable by 1

Getting data from the user

There exist three ways of getting data from the user using pure JavaScript.

  • alert(message): Shows the user the message, then waits until the user clicks "OK".
  • confirm(message): Shows the user the message, then waits until they click "OK" or "Cancel".
  • prompt(message, placeholder): Shows the user the message, allows them to write in a single-line text box that is pre-filled with placeholder, and waits until they click "OK" or "Cancel".

Flow Control: if()...else

Perform different actions if a condition evaluates to true or false.

var animal = "leopard";
if (animal === "leopard") {
    alert("The animal is a Leopard!");
} else {
    alert("The animal is not a Leopard!");
}

Flow Control: if()...else

if()...else can be chained with else if() 

var animal = "leopard";
if (animal === "leopard") {
    alert("The animal is a Leopard!");
} else if (animal === "tiger") {
	alert("The animal is a Tiger!");
} else if (animal === "panther") {
	alert("The animal is a Panther!");
} else {
    alert("The animal is something I don't know about!");
}

Logical Operators

  • && and 
    var animal = "leopard", wild = true;
if (animal === "leopard" && wild === true) {
	// This block will be executed only if animal is leopard and wild is true
	alert("The animal is a Wild Leopard!");
}
  • || or 
    var animal = "leopard";
if (animal === "leopard" || animal === "tiger") {
	// This block will be executed if animal is either leopard or tiger
	alert("The animal is a Leopard or Tiger!");
}
  • ! not 
    var isAdmin = false;
if (!isAdmin) {
	// This block will be executed only if isAdmin is set to false
	alert("You are not an Admin!");
}

The Math Object

  • Math.sqrt(number): Returns the square root of the given number.
  • Math.random(): Returns a random number between 0 and 1.
  • Math.round(number): Returns the next nearest integer.

The <noscript /> tag

  • The <noscript> tag is an HTML tag that will hide whatever content is inside of it when JavaScript is turned on, and show that content whenever it is turned off.
  • This is important, as every modern browser gives you the ability to turn off JavaScript.
    • In Firefox: Tools > Options > Content (Tab), uncheck "Enable JavaScript"
    • In Internet Explorer (7+): Tools > Internet Options > Security (Tab) > Custom level... > Scripting > Active Scripting, select "Disable"
  • By including the <noscript> tag and content inside, you can either alert people to turn on JavaScript, or show content that doesn't require JavaScript.

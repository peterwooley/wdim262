Homework: Final Project

Please turn in your files in a .zip file, using the following naming convention: lastnameFirstname_finalProject.zip

Final Project; due Week 11.

Requirements

Create a JavaScript Application that is either an Interactive Museum or Game. Make it fulfill these requirements:

  • The User Interface's behavior must be created in JavaScript.
  • The Style should be created in Cascading Stylesheets
  • The Content must be marked-up in (X)HTML
  • Visual design should be considered (and will be graded)
  • For an interactive museum, there must be at least five (5) "pages", which may or may not be separate .html files.
  • For a game, there must be a goal that the user can achieve (Not necessarily easily, but possible).
  • There must be at least three (3) JavaScript effects (Show/hiding elements, fading, animation, etc.)
  • There must be at least one (1) form with validation, and the content must be used in the App. (e.g., comment form, player name, etc.)

The <script> tag and you

    Where the <script> tag should go to be executed immediatly.

    <!DOCTYPE HTML>
<html>
	<head>
		<title>JavaScripting</title>
		<script type="text/javascript">
			// In the head tag
		</script>
	</head>
	<body>
	</body>
</html>

    Where the <script> tag should go to be executed after the HTML loads.

    <!DOCTYPE html>
<html>
	<head>
		<title>JavaScripting</title>
	</head>
	<body>
		<script type="text/javascript">
			// Just before the end of the body tag
		</script>
	</body>
</html>

	alert(data.items[0].media.m);
});

JSON — JavaScript Object Notation

JSON is a lightweight data-interchange format.

For us, it gets data from the server to the client and back again.

It is valid JavaScript.

It's simple for Server-side languages to parse and even simpler for JavaScript.

Here's a delicious sample:

{"menu": {
  "id": "file",
  "value": "File",
  "popup": {
    "menuitem": [
      {"value": "New", "onclick": "CreateNewDoc()"},
      {"value": "Open", "onclick": "OpenDoc()"},
      {"value": "Close", "onclick": "CloseDoc()"}
    ]
  }
}}
